## 一、第一次登录后台

### 步骤

1. 打开浏览器，访问：**https://yuexinys.cn/admin**
2. 输入默认账号：
   - 用户名：`admin`
   - 密码：`admin123`
3. 点击「登录」

> ⚠️ **登录后第一时间修改密码！** 见下一步。

## 二、修改后台密码（必须做）

### 步骤

1. 登录后台后，点击右上角 **「修改密码」**
2. 填写：
   - 当前密码：`admin123`
   - 新密码：建议 12 位以上，包含字母 + 数字 + 符号
   - 确认新密码：再次输入
3. 点击「确认修改」
4. **务必记住新密码**，建议保存在密码管理器里

> ✅ 修改成功后，下次登录请用新密码。

## 三、添加/编辑驿站信息

### 方式 A：后台手动添加（适合少量驿站）

1. 登录后台 → 左侧菜单点 **「驿站管理」**
2. 点击右上角 **「+ 新增驿站」**
3. 按表单填写（带 `*` 为必填）：

| 字段 | 怎么填 | 示例 |
|------|--------|------|
| 驿站名称 * | 写完整名称 | 深圳宝安青年驿站（西乡站） |
| 所在城市 * | 下拉选择 | 深圳 |
| 所在区域 * | 下拉选择 | 宝安区 |
| 详细地址 * | 复制粘贴官方地址 | 宝安区西乡街道XX路XX号 |
| 免费天数 | 查看官方政策填写 | 7 |
| 申请条件 | 逐条填写，每行一条 | 应届毕业生；深圳无房；… |
| 所需材料 | 逐条填写，每行一条 | 身份证；毕业证；… |
| 联系人 | 官方联系人姓名 | 李主任 |
| 联系电话 | 官方电话 | 0755-12345678 |
| 申请链接 | 官方申请网址（没有可不填） | https://... |
| 入住指引 | 富文本编辑器，可换行、加粗 | 步行路线、公交路线… |

4. 点击 **「保存」**
5. 保存后，点 **「获取经纬度」** 按钮（自动绑定高德地图位置）

### 方式 B：编辑已有驿站

1. 驿站管理列表 → 找到要修改的驿站 → 点 **「编辑」**
2. 修改对应字段 → 点 **「保存」**

### 方式 C：删除驿站

> ⚠️ 删除后无法恢复，请谨慎操作。

1. 驿站管理列表 → 找到要删除的驿站 → 点 **「删除」**
2. 弹窗确认 → 点 **「确认」**

## 四、批量导入驿站（Excel）

适合一次性导入几十上百条驿站数据。

### 第一步：下载模板

1. 后台 → 驿站管理 → 点 **「下载模板」**
2. 得到一个 Excel 文件

### 第二步：填写 Excel

> 📌 **不要修改表头**（第一行），只在下面行填数据。

各列填写说明：

| 列名 | 填什么 | 注意 |
|------|--------|------|
| 名称 | 驿站全名 | 必填 |
| 城市 | 广州/深圳/东莞… | 必填 |
| 区域 | 区/县名 | 必填 |
| 地址 | 详细地址 | 必填，影响地图定位 |
| 免费天数 | 数字 | 没有填 0 |
| 申请条件 | 分号分隔 | 示例：应届毕业生;无房 |
| 所需材料 | 分号分隔 | 示例：身份证;毕业证 |
| 联系人 | 姓名 | 选填 |
| 电话 | 电话号码 | 选填 |
| 申请链接 | 网址 | 选填 |

### 第三步：上传导入

1. 后台 → 驿站管理 → 点 **「导入 Excel」**
2. 选择填好的 Excel 文件 → 点 **「上传」**
3. 系统提示「成功导入 N 条」即完成

> ❓ 如果导入失败：检查 Excel 里是否有空行、城市名是否拼写正确。

## 五、生成驿站码发给学校

驿站码是学生登录网站的「通行证」，通过学校/机构分发给学生。

### 生成驿站码

1. 后台 → 左侧菜单点 **「驿站码管理」**
2. 点 **「批量生成」**
3. 填写：
   - 生成数量：建议 50~200（每所学校一批）
   - 使用次数上限：建议 100（防止被滥用）
   - 有效期：建议设到当年 12 月 31 日
   - 备注：填写学校名称，方便管理（如「华南理工大学 2026」）
4. 点 **「生成」**
5. 生成后点 **「下载 CSV」**，得到一个表格文件

### 把驿站码发给学校

1. 打开下载的 CSV 文件（用 Excel 或 WPS）
2. 将「驿站码」那一列复制出来
3. 通过微信/邮件发给对应学校的就业办老师
4. 附上说明文字（可复制下方模板）：

---
**发给老师的参考话术：**

> 老师好！这是「青年驿站」平台的登录码，请分发给有需要的应届毕业生。
> 使用方法：访问 https://yuexinys.cn → 点击「驿站码登录」→ 输入登录码即可。
> 每个码最多可使用 XXX 次，有效期至 XXXX 年 XX 月 XX 日。

## 六、更新政策信息

### 什么时候需要更新？

- 政府发布了新的人才补贴政策
- 原有政策的申请条件有变化
- 链接失效了

### 操作步骤

1. 后台 → 左侧菜单点 **「政策管理」**
2. 选择城市（如「深圳」）
3. 点 **「编辑」** 修改现有政策，或点 **「新增」** 添加新政策
4. 填写：
   - 政策名称（如「深圳市新引进人才租房补贴」）
   - 补贴金额
   - 申请条件
   - 官方申请链接
5. 点 **「保存」**

## 七、管理 AI 知识库

AI 助手「湾湾鲸」的答案质量，取决于知识库里有没有相关资料。

### 什么时候需要更新知识库？

- 有新政策文件发布
- 学生频繁问某个问题，但 AI 回答不准确
- 驿站信息有重大更新

### 方式 A：上传文档（PDF/Word/TXT）

1. 后台 → 左侧菜单点 **「知识库管理」**
2. 点 **「上传文档」**
3. 选择文件（PDF、Word、TXT 均可）
4. 填写标题（如「2026年深圳市人才补贴政策」）
5. 点 **「上传并解析」**
6. 等待系统提示「解析完成」

### 方式 B：添加网页链接

1. 知识库管理 → 点 **「添加链接」**
2. 粘贴政府官网的政策页面链接
3. 填写标题
4. 点 **「抓取并解析」**

### 方式 C：直接写笔记

> 适合把常见问题答案直接录入。

1. 知识库管理 → 点 **「写笔记」**
2. 标题：写问题（如「申请驿站需要什么材料？」）
3. 内容：写详细答案
4. 点 **「保存」**

### 查看 AI 回答是否准确

1. 前台打开网站 → 点右下角海豚图标打开 AI 助手
2. 问几个常见问题，检查回答质量
3. 如果回答不准确，按上述方式补充知识库

### 怎么看数据

1. 后台 → 左侧菜单点 **「数据看板」**
2. 可以看到：
   - **总访问量（PV）**：页面被刷新了多少次

## 九、常见问题排查

### 问题 1：学生说「驿站码无效」

**可能原因**：
- 驿站码已过期（检查「驿站码管理」里的有效期）
- 使用次数用完了（看「已使用次数」是否达到上限）
- 驿站码被禁用了（状态应为「启用」）

**解决方法**：重新生成一批新码，发给学校。

---

### 问题 2：地图上找不到某个驿站

**可能原因**：地址填写不准确，高德地图无法定位。

**解决方法**：
1. 驿站管理 → 编辑该驿站
2. 检查「详细地址」是否完整（建议加上城市+区域）
3. 点「获取经纬度」重新定位
4. 如果仍失败，手动在高德地图网站搜索该地址，复制坐标填入

---

### 问题 3：网站打不开

**第一步**：用自己的手机 4G 网络访问 https://yuexinys.cn，确认是否只是自己公司网络问题。

**第二步**：如果所有人都无法访问，联系技术负责人（Falgan）。

**紧急情况处理**：
- 在微信搜索「青年驿站」是否有备用群
- 在 GitHub 仓库提 Issue：https://github.com/Falgan678/youth-station/issues

---

### 问题 4：忘记后台密码

联系技术负责人（Falgan）重置密码。**不要反复尝试**，会导致账号被锁定。
